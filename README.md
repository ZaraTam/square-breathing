# Square breathing<br>
<br>
Square breathing technique (aka box breathing technique) is a form of relaxation training.<br>
<br>
Cycle<br>
1. N counts in<br>
2. N counts hold<br>
3. N counts out<br>
4. N counts hold<br>
<br>
See [demo](https://zaratam.github.io/square-breathing/)<br>
where N = `duration` = 4 seconds
